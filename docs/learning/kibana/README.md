## üîó Quick Reference

### **Essential Navigation**
```
Kibana Home ‚Üí Analytics ‚Üí [Your App]
- Discover: Interactive data exploration
- Lens: Drag-and-drop visualization builder
- Dashboard: Combine visualizations
- Canvas: Custom presentations
- Maps: Geospatial visualization
```

### **Key Concepts**
- **Data View**: Configuration connecting Kibana to Elasticsearch indices
- **Lens**: Modern drag-and-drop visualization builder
- **Canvas**: Pixel-perfect presentation tool
- **Space**: Multi-tenant workspace for organizing content
- **KQL**: Kibana Query Language for filtering data

### **Common UI Patterns**
- **Field Lists**: Drag fields to create visualizations
- **Time Picker**: Control date ranges across all visualizations
- **Filters**: Add context-aware filters to dashboards
- **Save/Share**: Save work and share with teams
